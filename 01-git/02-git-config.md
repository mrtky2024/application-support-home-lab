
# Git Config(Setting commit author info)

## 1. Set Git Username (Set the author name for commits)
git config --global user.name "Murat Kaya"
---
## 2. Set Git Email (Set the author email for commits)
git config --global user.email "abc@mail.com"
---
## 3. Verify Git Configuration (Check current Git configuration)
git config --global --list
